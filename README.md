# Sistema Bancário Simples em Python

Este projeto consiste em um **Sistema Bancário Simples** desenvolvido em Python como parte de um desafio da DIO (Digital Innovation One). O objetivo é aplicar conceitos básicos de lógica de programação, estruturas de repetição e condicionais na simulação de operações bancárias do dia a dia.

---

## 🚀 Funcionalidades

O sistema permite que um único usuário realize três operações essenciais:

* **Depósito:** Permite adicionar fundos à conta. Todos os valores depositados devem ser positivos e são registrados no histórico do extrato.
* **Saque:** Permite retirar fundos respeitando os seguintes limites:
    * Máximo de **3 saques diários**.
    * Limite máximo de **R$ 500,00 por saque**.
    * O usuário deve ter saldo suficiente em conta.
* **Extrato:** Lista todos os depósitos e saques realizados. No fim da listagem, exibe o saldo atualizado da conta utilizando a formatação `R$ XXX.XX`.
